1. 将所有头文件和cpp文件编译成为exe文件后（这里需要mnist库，放到当前cpp文件目录即可)，生成*.exe文件
2. 进入*.exe执行文件的目录后，在命令行下敲击命令，比如敲击*.exe -t 10 -r 0.05 -b 10，其中-t代表训练次数，-r代表学习率，-b代表批的个数，训练次数默认为5，
   学习率默认为0.01，批默认为1
3. 当敲击命令后，系统会在data文件夹下读取result.txt文件，其中result.txt储存了原来神经网络训练的参数，然后再进行训练，最后将参数写回result.txt
4. 利用result.txt测试生成的label.txt会保存在data文件夹下，其中label.txt代表各个数字测试的结果
